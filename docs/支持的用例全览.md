## 四、交易生命周期状态

| # | 用例 | 示例查询 | SQL 逻辑 |
|---|------|---------|----------|
| 24 | 未到期交易 | 浙江分公司未到期交易量 | JOIN XF_FX_TOTALDELIVERY，未交割金额 > 0 且 MATURITYDATE > SYSDATE |
| 25 | 逾期交易 | 浙江分公司逾期交易量 | 未交割金额 > 0 且 MATURITYDATE < SYSDATE |
| 26 | 已到期交易 | 浙江分公司已到期交易量 | 未交割金额 > 0 且 MATURITYDATE = SYSDATE |
| 27 | 未完结交易 | 浙江分公司未完结交易量 | 未交割金额 > 0（未到期 + 逾期 + 已到期） |
| 28 | 已完结交易 | 浙江分公司已完结交易量 | 未交割金额 = 0 |

> 关键词：未到期 / 逾期 / 已到期 / 未完结（未结清、未了结）/ 已完结（已结清、已了结）

## 九、金额筛选

| # | 用例 | 示例查询 | 说明 |
|---|------|---------|------|
| 44 | 金额大于 | 浙江分公司本月交易量大于 100 万美元的客户 | HAVING SUM > value |

## 十二、上下文继承（多轮对话）

| # | 用例 | 第 1 轮 | 第 2 轮（追问） | 继承项 |
|---|------|--------|----------------|--------|
| 50 | 追问买卖方向 | 浙江分公司本月交易量 | 其中买入的呢 | bank_name, date, product_type |
| 51 | 追问生命周期 | 浙江分公司本月远期交易量 | 其中逾期的有多少 | bank_name, date, lifecycle_status |
| 52 | 追问对比 | 浙江分公司本月交易量 | 同比呢 | 全部参数 + comparison=yoy |
| 53 | 追问换维度 | 浙江分公司本月交易量 | 按客户看 | bank_name, date, dimension |
| 54 | 追问排名 | 浙江分公司本月交易量 | TOP10 | bank_name, date, top_n=10 |

> 继承策略：框架参数（bank_name, cust_name, product_type, date, dimension 等）始终继承；查询特定参数（special_states, lifecycle_status, trade_class）仅在追问时继承。追问检测基于关键词模式（呢/换/还/同比/环比/维度/排名等）。

## 附录：参数维度交叉表

| 维度 | 可选值 | 来源 |
|------|--------|------|
| 产品类型 | all / spot / fwd / swap | 关键词匹配 |
| 买卖方向 | B(买入/结汇) / S(卖出/购汇/售汇) / 空 | 关键词 + 结售汇铁律 |
| 时间区间 | 12+ 种表达式 | 正则 + 日期计算 |
| 机构名称 | 任意中文机构名 | 正则提取 + XF_BASE_BANK 模糊匹配 |
| 客户名称 | 任意中文客户名 | 正则提取 + CUSTNAME 精确匹配 |
| 生命周期 | not_due / overdue / due_today / unclosed / closed | JOIN XF_FX_TOTALDELIVERY |
| 特殊状态 | 0 / 1,2,6,7,10,11,15,17 / 3,5,12,13 / 4,16 | SPECIALSTATE 字段 |
| 聚合维度 | bank / customer / customer_id / manager / manager_name / product_type / month | GROUP BY 列 |
| 排名 | TOP 1~100 | ROWNUM 截断 |
| 套保率 | 衍生品金额占比 | CASE WHEN PT IN ('fwd','swap') |
| 金额筛选 | gt / gte / lt / lte + 数值 | HAVING 子句 |
| 对比 | yoy(同比) / mom(环比) | 日期偏移计算 |
| 业务系统 | 1(外汇) / 2(结售汇) | APPID 字段，结售汇关键词自动设置 |

---

**共计 60+ 种用例**，覆盖 产品类型 × 买卖方向 × 时间维度 × 生命周期 × 特殊状态 × 聚合维度 × 排名 × 套保率 × 金额筛选 × 同比环比 的排列组合。
